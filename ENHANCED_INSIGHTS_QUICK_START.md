# Quick Start - Enhanced AI Trading Insights

## What's New? ✨

Po dokončení **Morning Checku** se nyní automaticky generují **komplexní AI insights** s:
- 📊 Trading Readiness Score (0-100%)
- 🚨 Kritické upozornění na psychologické vzory
- 📈 Performance analýza z tvých recentních tradů
- 🧠 Psychologické detekční (revenge trading, FOMO, atd.)
- ⏰ Market timing recommendations
- ✅ Interactive pre-trade checklist

## How It Works

### 1. Fill Morning Check ✅
Vyplň denní Morning Check se všemi parametry:
- Sleep quality & hours
- Energy level
- Stress level
- Focus
- Emotional state
- Meditation & routines

### 2. Insights Auto-Generate 🤖
Jakmile je Morning Check vyplněn:
- Systém automaticky načte tvou historii tradů
- Analyzuje psychologické patterny
- Porovnává s historickým normálem
- Generuje 6-8 personalizovaných insights

### 3. Review Your Insights 📋
Insights se zobrazují v **Daily Tracker** pod Morning Check sekcí:

```

│ 📊 Trading Readiness Score      75%     │
│ ████████░░░░░░░░░░░░░░░░░░░░░░        │
│ "Akceptabilní. Zvýšená opatrnost."     │
└─────────────────────────────────────────┘

Tab Interface:
[ Všechny ] [ Ranní ] [ Výkon ] [ Psychology ] ...

Critical Alerts:

   → Medituj 5 minut před trading

Insights by Category:

```

## Insight Types

### 🔴 CRITICAL (Act Now!)
- Stres > 7/10 → Risk revenge trading
- Sleep < 6h → Degraded decision-making
- Pattern: Revenge trading detected

### 🟠 HIGH (Be Careful)
- Energy < 4 → Slow reactions
- Win rate < 40% → Review setup
- Readiness < 50% → Consider demo mode

### 🟡 MEDIUM (Note)
- Focus issues → Distractions present
- Friday → Higher volatility
- Early exit pattern → Profit taking too early

### 🟢 LOW (Good News!)
- Great win rate (>60%) → Keep it up!
- Confident mindset → Flow state
- Excellent week → Momentum!

## Action Buttons

Některé insights mají actionable buttons:

- **"Meditovat 5 min"** → Integrace s timer
- **"Zvýšit energii"** → Hydration/exercise reminder
- **"Analyzovat selhavé trades"** → Link na fail log
- **"Přejít na demo"** → Switch to virtual trading

## Priority System

Insights jsou seřazeny podle priority:
1. **CRITICAL** (Red) - Immediate action
2. **HIGH** (Orange) - Important consideration
3. **MEDIUM** (Yellow) - Useful information
4. **LOW** (Green) - Positive feedback

## Readiness Score Interpretation

```
🟢 80-100% "Ideální podmínky pro trading"
   → Perfektní čas pro A+ setupy s plnou silou

🟡 60-79% "Dobré podmínky s výhradami"
   → Trading je OK, ale zvýšená opatrnost

🟠 40-59% "Rizikové"
   → Zvažit zkrácení dne nebo demo trading

🔴 0-39% "Není připraveno"
   → Zvažit den mimo trading
```

## Data Sources for Insights

Systém analyzuje:

| Source | Insights |
|--------|----------|
| Morning Check | Sleep, stress, energy, focus, emotions |
| Recent Trades (last 20) | Win rate, revenge trading, early exits |

## Troubleshooting

**Q: Insights nejsou se generují?**
- Zkontroluj morning check je vyplněn
- Zkontroluj je uložen do databáze
- Refreshni stránku

**Q: Insights se nezdají relevantní?**
- Potřebují více tradů v historii
- Zkontroluj trading identity je vyplněna
- Zkontroluj morning check data je správná

**Q: Loading se trvá dlouho?**
- Trvá 2-5 sekund (normální)
- Check internet connection
- Zkontroluj browser console pro errors
